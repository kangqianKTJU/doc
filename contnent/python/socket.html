socket:



该模块提供对BSD套接字接口的访问。它适用于所有现代Unix系统，Windows，MacOS以及其他可能的平台。

Python接口是Unix系统调用的直译，翻译成python面向对象式的套接字接口库：socket（）函数返回一个套接字对象， 其方法可以实现各种套接字系统调用。 参数类型比C接口更高级：与Python文件操作中的read（）和 write（）函数一样，接收操作上的缓冲区分配是自动的，缓冲区长度隐含在发送操作上。

socket族



基于不同的系统和构建选项，此模块提供不同的套接字族。

根据创建套接字对象时指定的地址系列，自动选择特定套接字对象所需的地址格式。 套接字地址表示如下：

  • AF_UNIX: Unix中同一台机器进程间通信协议
  • AF_INET: IPv4协议
  • AF_INET6:IPv6协议
  • AF_NETLINK: 内核态和用户态进程之间进行数据传输的特殊的IPC
  • AF_CAN: CAN总线
  • AF_TIPC:TIPC协议
  • AF_BLUETOOTH:蓝牙协议
  • AF_ALG: 基于Linux的基于套接字的内核加密接口
  • AF_VSOCK: 虚拟机与主机通讯

常量



基于不同的系统和构建选项，此模块提供不同的套接字族。

根据创建套接字对象时指定的地址系列，自动选择特定套接字对象所需的地址格式。 套接字地址表示如下：

  • socket.AF_UNIX
  • socket.AF_INET
  • socket.AF_INET6
  • socket.SOCK_STREAM
  • socket.SOCK_DGRAM
  • socket.SOCK_RAW
  • socket.has_ipv6
  • ...

方法



socket.Socket(family=AF_INET, type=SOCK_STREAM, proto=0, fileno=None)

创建新套接字：传入协议族、套接字类型、协议号；也可以从文件读取。默认的协议是IPV4协议， 默认的套接字类型是SOCK_STREAM,协议号一般是0。如果 fileno 指定了,协议族、类型、协议号会 从指定的文件描述符中检测出来。 

	import socket
	s = socket.socket()
	s
	<socket.socket fd=492, family=AddressFamily.AF_INET, type=SocketKind.SOCK_STREAM, proto=0>
socket.socketpair([family[, type[, proto]]])

使用给定的地址族、类型和协议号，构建一对连接的套接字对象。 地址族，套接字类型和协议号与上面的 socket（）函数相同。 如果在平台上定义，则默认系列为AF_UNIX; 否则，默认为AF_INET。 

	import socket
	s = socket.socketpair()
	s
	(<socket.socket fd=468, family=AddressFamily.AF_INET, type=SocketKind.SOCK_STREAM, proto=0, laddr=('127.0.0.1', 8848), raddr=('127.0.0.1', 8849)>,
	 <socket.socket fd=488, family=AddressFamily.AF_INET, type=SocketKind.SOCK_STREAM, proto=0, laddr=('127.0.0.1', 8849), raddr=('127.0.0.1', 8848)>)
socket.create_connection(address[, timeout[, source_address]])

连接到侦听Internet地址（2元组（主机，端口））的TCP服务，并返回套接字对象。这是一个比socket.connect（）更高级的函数： 如果host是非数字主机名，它将尝试为AF_INET和AF_INET6解析它，然后尝试依次连接到所有可能的地址，直到连接成功。这样可以轻松编写与IPv4和IPv6兼容的客户端。

如果传递可选的timeout参数,那么将在尝试连接之前设置套接字实例上的超时。 如果未提供超时，则使用getdefaulttimeout（）返回的全局默认超时设置。

如果有source_address的话，则必须是要连接的套接字的2元组（主机，端口）作为其源地址才能连接。 如果主机或端口分别为''或0，则将使用OS默认行为。
