Abaqus简介

话不多说，Abaqus是一款有限元模拟软件。

产品示意图如下：




  • Abaqus / Standard：
    一款通用分析产品，可以解决涉及元件静态，动态，热，电和电磁响应的各种线性和非线性问题。
  • Abaqus / Explicit：
    一款特殊用途的分析产品，它使用显式动态有限元公式，适用于模拟短暂的瞬态动态事件，如冲击和爆炸问题，且对于涉及改变接触条件的高度非线性问题（如形成模拟）也非常有效。
  • Abaqus / CFD：
    是一种计算流体动力学分析产品。 它可以解决一大类不可压缩的流动问题，包括层流和湍流，热对流流动和变形网格问题。
  • Abaqus / CAEd：
    （完整的Abaqus环境）是Abaqus的交互式图形环境。 它允许通过生成或导入要分析的结构的几何形状并将几何体分解成可网格区域来快速且容易地创建模型。 物理和材料属性可以与载荷和边界条件一起分配给几何体。 Abaqus / CAE包含非常强大的选项，可以对几何体进行网格化并验证生成的分析模型。 模型完成后，Abaqus / CAE可以提交，监控和控制分析作业。
  • Abaqus / Viewer
    Abaqus / CAE的子集，仅包含Visualization模块的后处理功能。
Abaqus接口与CAE：




可见，Abaqus最终都会生成*.inp类型的文件，最后提交给求解器进行求解。

GUI接口(图形界面)、命令行接口、脚本接口都会生成*.py类型的文件，交由python解释器解释执行。执行的过程 Abaqus会以*.rpy类型的文件记录下操作的每一步，其实就是python文件。最后python解释器生成*inp文件，交由求解器求解，求解出的结果写入数据库文件，然后可以通过，Viewers工具可视化结果。

Abaqus接口操作
  1. 对*.inp文件操作：

    2. Abaqus中，*.inp文件是作业提交文件，是Abaqus求解器唯一能识别的文件，提交*inp是最快的计算方法。 在Abaqus命令行下提交*inp文件的方法如下： 

     abaqus job = 2DDoubleEdSymmCPE8R_model

    然后在temp目录下就会得到输出的结果信息文件。

    Abaqus提供了很多的计算案列，在他的帮助文档里都有详细的介绍，下面是我的帮助例子所在的路径：

    C:\SIMULIA\Abaqus\6.14-4\samples\job_archive\samples

    Abaqus提供了在命令行下直接获取案例文件的命令fetch,通过该命令可以直接将案例文件提取到工作路径下。

    abaqus fetch job=2DDoubleEdSymmCPE8R_model.inp    //获取2DDoubleEdSymmCPE8R_model.inp文件
    abaqus fetch job=2DDoubleEdSymmCPE8R_model        //获取名字为2DDoubleEdSymmCPE8R_model相关的文件,不仅仅是inp文件，还可能是py文件、f文件
  2. 对*.py文件操作：

    3. 在命令行下输入： 

    abaqus cae script = DDoubleEdSymmCPE8R_model.py      //以CAE图形界面打开模型，cae也可以用viewer代替
    abaqus cae noGUI=myscript.py                           //以非图形界面建模 ，cae也可以用viewer代替

    python文件另一大好处就是，你可以把它在CAE界面的CLI栏里一条条的输入，这样你就可以了解模型构建的每一步了，这对于学习Abaqus的例子来说比单纯的Cae文件要好多了。
