SubModel技术

Abaqus的子模型技术（submodel）是在完整模型的基础上建立起来的。

  1. 首先使用较粗糙的几何体和网格建立全局模型。
  2. 将全局模型的数据导入子模型，或者将全局模型的结果导入子模型。
1.创建子模型

全局模型分析完后，得到数据库或者是结果文件，就可以创建子模型了。

首先，将全局模型复制到新模型（即子模型）。

Model→Copy→global model name→submodel name

然后将全局模型的结果文件导入到子模型中

Model→Edit Attributes→submodel name

勾选上Read data from job,输入全局模型结果文件名，可以是.odb .fil文件

2.移除区域

Remove regions from the submodel that you are not interested in analyzing. Only the regions of interest should remain. You can use several techniques to remove regions from a part.
